# 🎯 Machine Gun Balloon Shooting Game

(https://via.placeholder.com/600x400?text=Game+Demo+Screenshot)

A fun shooting game developed with HTML5, CSS3, and JavaScript. Players need to control a machine gun to shoot rising balloons and score as many points as possible within the time limit.

## 🎮 Live Demo

[Click here to play the game](https://lawson-dong.github.io/Vibe-coding-fun-games)

## ✨ Features

- 🎯 Mouse-controlled machine gun aiming, hold left button for continuous shooting
- 🎈 Colorful balloons randomly generated, floating upward
- ⏱️ 30-second time challenge to test your reaction speed
- 💯 Real-time score and countdown display
- 🎨 Beautiful visual feedback (balloon explosion effects)
- 📱 Responsive design, supports multiple screen sizes

## 🎯 How to Play

1. Click the **"Start Game"** button
2. Move the mouse to aim the machine gun left and right
3. **Hold the left mouse button** to fire bullets
4. Hit balloons to score 1 point
5. Score as many points as possible within 30 seconds!

## 🛠️ Tech Stack

- **HTML5** - Game structure
- **CSS3** - Styling and animations
- **JavaScript** - Game logic and interactions
